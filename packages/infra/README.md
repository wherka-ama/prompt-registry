# @prompt-registry/infra

Infrastructure layer for Prompt Registry - GitHub clients, harvesters, search, and storage.

## Installation

```bash
npm install @prompt-registry/infra
```

## Usage

This package provides infrastructure implementations for:
- GitHub API clients and token providers
- Bundle harvesters from GitHub and other sources
- Search and indexing
- Storage and state management
- File system and HTTP utilities
